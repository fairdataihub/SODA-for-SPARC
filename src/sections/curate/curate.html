Step 2: Specify high-level folders

Which SPARC high-level folders
would you like to add to your dataset?
Note: You will be able to delete undesired folders in step 3.
Please select at least one:

code

Folder containing all the source code used in the study

primary

Folder containing all folders and files for experimental subjects and/or samples

source

Folder containing very raw data i.e. raw or untouched files from an experiment

docs

Folder containing all other supporting files that don't belong to any of the other folders

protocol

Folder containing supplementary files to accompany the protocols submitted to Protocols.io.

derivative

Folder containing data files derived from raw data
