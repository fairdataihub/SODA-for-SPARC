Contributor information

Provide information about the individuals who contributed to this dataset. These individuals do not have to be authors of any publications that describe the data, but they should be acknowledged for their role in producing and publishing the dataset.

README information

All SPARC datasets must include a README file. This is a plain text file that contains necessary details for the reuse of the data beyond that which is captured in the other metadata file. Some information that should be included is:

  • How would a user use the files that are provided? e.g., first open file X and then look at file Y.
  • What additional details do users need to know? Are some subjects missing data?
  • Are there warnings about how to use the data or code?
  • Are there appropriate/inappropriate uses for this data?
  • Are there other places that users can go for more information? e.g., did you provide a GitHub repository or are there additional papers beyond what was provided in the metadata form?

You can provide that information below. SODA will use that information to automatically create the README metadata file and include it in your dataset when it is generated in the last step of this guided workflow.

Dataset permissions

Dataset permissions control who can view, edit, or manage your dataset on Pennsieve before it is published. You can give permissions to your team members if you expect them to modify your dataset before publication.

Learn more about dataset Permissions on Pennsieve

Pennsieve has 3 levels of permissions for datasets:

  • Viewer: Give this permission to collaborators who only need to view your dataset.
  • Editor: Give this permission to collaborators who only need to view your dataset and upload files to it.
  • Manager: Give this permission to collaborators who need full editing access to your dataset (except publish the dataset).
  • Owner: This permission is typically given to the Principal Investigator of the award. The owner can do everything with the dataset, including publish it.

Current permissions are listed in the table below. To add a permission, select the Pennsieve user or team you would like to grant a permission to, select their role, and press "Designate permission". To delete a permission, click the delete icon in the table below.

Editing this dataset

If you need to edit this dataset before sharing it with the curation team, you may do so by clicking either "Free Form Mode" or "Continue working on a dataset" on the home page. Selecting "Free Form Mode" is best for making small, individual changes to your dataset, while "Continue working on a dataset" will take you through the same end to end process as the initial upload but will allow you to edit your data uploaded to Pennsieve. Once your dataset is ready for SPARC curation review, you may share it with the curation team below.

Reserve a DOI

Reserve a DOI for your dataset while it's being reviewed by the curation team (for instance, if you need to refer to this dataset in a manuscript right away). Note: This DOI will NOT be active, and therefore searchable, until the dataset is published to Pennsieve Discover. If you send your dataset to the curation team without reserving a DOI you will not be able to reserve one after due to the dataset being locked.
