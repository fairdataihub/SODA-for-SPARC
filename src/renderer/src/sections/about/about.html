About SODA

Streamlining FAIR Data Sharing

SODA (software to organize data automatically) is a cross-platform desktop software that helps researchers prepare and share FAIR peripheral nervous system (PNS) related data and models using the SPARC Data Structure (SDS) and the SPARC Portal. Since 2021, SODA has been installed on over 1300 computers, empowering researchers worldwide.
