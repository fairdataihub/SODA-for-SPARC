Create manifest.xlsx

Need help?

The SPARC guidelines require all datasets to include metadata files called manifest files. One manifest file must be included in each high level folder. Manifest files are intended to contain information about all of the data files in its high level folder. You can request SODA to generate manifest files automatically for you.
