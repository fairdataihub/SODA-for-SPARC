What do these options do?

Tell SODA how to handle folders specified during Step 3 that already exist (same name, same relative location) on your Pennsieve dataset:

  • Replace: Delete existing folders on Pennsieve then upload new ones
  • Merge: Upload new files into the existing folders. Existing files will be handled as intructed in the next step.
  • Skip: Existing folders will not be uploaded.
