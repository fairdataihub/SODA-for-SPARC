Would you like SODA to auto-generate your manifest file?

The manifest file is a metadata file that contains a list of all of the folders and files in your dataset along with information about them. SODA can automatically generate it for your dataset.

The selected Pennsieve dataset already includes data files. To prevent conflicts, SODA will not generate manifest files. If you imported a manifest file with your dataset SODA will upload that as is.
